FatturaREST JAX-RS

Endpoint richiamabili dal browser (GET)

  • Lista fatture, paginata
    rest/fatture?from=1&to=3
  • Lista fatture, filtrata
    rest/fatture?pIVA=123
  • Numero fatture
    rest/fatture/count
  • Numero fatture per partita IVA
    rest/fatture/count?pIVA=123
  • Lista fatture di un anno
    rest/fatture/2020
  • Fattura completa
    rest/fatture/2020/1
  • Dettaglio partita IVA per fattura
    rest/fatture/2020/1/pIVA
  • Dettaglio attachment binario per fattura
    rest/fatture/2020/1/attachment
  • Dettaglio elementi per fattura
    rest/fatture/2020/1/elementi
  • Dettaglio elemento per fattura
    rest/fatture/2020/1/elementi/2
  • Lista fatture per codice prodotto
    rest/prodotti/12abc/fatture
